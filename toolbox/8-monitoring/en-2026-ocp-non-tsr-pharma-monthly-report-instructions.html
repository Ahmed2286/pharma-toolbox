• MONTHLY PHARMACY ANALYSIS REPORT

HOW TO FILL IT IN
(Version February 2026)

INTRODUCTION

The report is needed to:

•
Help the pharmacy responsible to structure his analysis and make existing management
problems visible to take appropriated decisions.
•
Help the coordination to analyze the pharmacy management, be alerted on time and follow
the project activity.
•
Help the HQ to do a macro analysis, improve the support and capitalize the information.
•
Have a written proof of the pharmacy activity in the project – it may be requested in case of
an external audit.

For these reasons, the monthly pharmacy analysis report is mandatory.

It must be filled in by the person in charge of the project pharmacy and must be sent with the last
backup from Isystock, the valued inventory report and all the annexes (sleeping stock, overstock,
rupture, pre-rupture, pre-expired, new articles) to the CoPharma/Country Pharma with the MedCo in
copy – latest the 07th of the month.

PREREQUISITES

Having an updated Isystock that means:

•
All movements of the last month are recorded; ensure that there is no negative value in the
initial stock and/or the theoretical stock.
•
The inventory (complete or partial) is recorded.
•
The sub folder of the most recent filters of the Isystock folder is available on the computer.
(After the use of the filter, always check if data extracted from Isystock are those expected)

IMPORTANT CONCEPTS TO CONSIDERE

-
Project stock managed at coordination level: this stock should be taken in consideration. The
coordination team must send at the end of the month the situation of project stock managed
at their level.

-
Update attributes: "consumable" and "non-consumable" attributes in Isystock are for
information only, it is advisable for each project to adapt these attributes to the realities of
the field.

PROJECT INFORMATION SHEET

General project pharmacy related information which tends to be stable, thus there is no need to
update it monthly, only when there are changes in that information.

PROJECT CODE and ISYSTOCK CODE
The Isystock code identifies the project pharmacy and is composed of the project code (the two ID
letters of the country + the figures of the budget code) plus three letters.
Example CD146RUT = CD (DRC) + 146 (Rutshuru project) + RUT (Rutshuru pharmacy)

The code is visible in the headline of the Isystock.
If the code does not correspond to the mentioned rule, contact your Isystock referent in Paris for an
update.

CURRENCY USED
All article prices must be in the same currency, in general in Euro. Depending on the country, another
currency can be used after discussion with the Isystock referent in Paris and the financial
department.

STOCK VALUE AT THE END OF THE PREVOUIS YEAR
The value at the end of previous year will be used to make a matching of all movements at the end of
the current year.

HUMAN RESSOURCES
A brief description on the human resources working on the pharmacy: name/ position/ job
description (up to date & signed?)/ level of experience / seniority/trainings, etc.

Good storage practices have a direct impact on medical items quality. It is important to identify if
they are correctly in place, and to define action points if need it. Wrong GSP have a negative impact
on stock management.

Storage capacity is important to respect GSP. If a clear storage capacity/need is not available, you can
contact the pharma referent who will in collaboration with logistic and supply departments provide
the support needed.

MONTHLY/QUARTERLY OPTION
By answering the YES / NO question in cell F9, the section that is to be filled quarterly would appear
or disappear.

QUARTERLY/MONTHLY NOTION

The monthly report is now composed of 2 main sections: a section that is to be filled on a quarterly
basis and a section that is to be filled monthly. To be able to use this option please to activate the
content after opening the file.

-
Quarterly chapters: it is advisable to complete these chapters quarterly, but if a project wish
to fill them monthly, they still can. Quarterly data must consider all movements since the last
analysis. Example: if the last analysis was done in July, and now you are doing one in
October, the one of October must consider the IN / OUT movements, the losses and all other
the movements from July to October. Quarterly analysis of the Month of December is always
mandatory, because it is used to complete the annual report.

-
Monthly chapters: no change compared to other years.

SLEEPING STOCK

Use the report "monthly statement: simplified one" of the month with a CMM of 12 months. (If the
project has been open for less than 12 months, select the number of months the project has been
open).
Apply the filter "no conso but stock.xml". (See Manual Isystock 2012 or later) and export to excel.
Save as xls.
Be careful, the order of the columns needs to be the original standard order (see below).

Add at the end of the column "valeur_tot" a formula to calculate the sum of the column:
= SUM(N2:Nxxx)
Fill in the number of articles and the value.

Be careful, this figure must be detailed per products group; otherwise, it gives a false impression of
the real sleeping stock. There is indeed a difference between articles considered as "consumables"
(regular consumption) and articles considered as "non-consumable" (irregular consumption such as
for equipment).

To fill in the repartition of sleeping stock by products group:
Apply to the xls table the automatic filters (Data -> and then click on the “Filter” button) on the
headline of the columns.
Find the value per group of items using standard xls filters (filter on the column “conso”).
Use a SUM formula to get the total value for each category.

OVERSTOCK

Use the report "monthly statement of the stock pharmacy: simplified one" of the month with a CMM
of 12 months (If the project has been open for less than 12 months select the number of months the
project has been open).
Apply the filter « overstock.xml » (See Manual Isystock 2012 or later) and export to excel.
Save as xls.
Attention: the order of the columns needs to be the original standard order (see below).

Keep in mind that the value of over stock will be the value above 12 months. Meaning the month
of consumption that are above what you need for 12 months.
E.g.: If your stock of Paracetamol 500 mg is 25 months of consumption, you need to calculate the
value of 13 months and not of the whole 25 months.
The formula to calculate this value is:
(Months of stock AMC - 12)*AMC*(Total Value/Final Stock)
Add in the first empty column of the table the above formula.
Then add at the end of this column the formula to calculate the sum of the column:
=SUM(X2;Xxxx)
Fill in the number of articles and the value.
To fill in the repartition of over stock by products group:
Apply to the xls table the automatic filters (Data -> and then click on the “Filter” button) on the
headline of the columns.
Find the value per group of items using standard xls filters (filter on the column “conso”). Use a SUM
formula to get the total value for each category.

LOSSES
Material damaged after use for a normal period in a ward is not considered as damaged and not to
enter in Isystock.
Use the report « Losses report » for the period. Do not tick “all units”.
Export to excel.
Save as xls.
Apply to the xls table the automatic filters (Data -> and then click on the “Filter” button) on the
headline of the columns.
Filter on the column « OUT_Expired items »
Count the number of lines to have the number of expired items.
(Count the number of different items and not the number of lines.)
Then calculate the sum of the « Total value » column to have the total value of expired items.

For « Damaged» and « Cold Chain Breaks » losses, repeat these actions for the « OUT_Damaged
items » and « OUT_Cold Chain Break » columns.

For « Any others » Losses in the report, repeat these actions for the « OUT_Rappel de lot »,
« OUT_Vol » and « OUT_Autres pertes » columns.

THEORETICAL STOCKS AND PHYSICAL STOCK (INVENTORY)

Each month an inventory must be done (complete or partial – see procedure « inventory »).

INVENTORY
Complete inventory: All items are counted.
Partial inventory: At least 10% of the number of referenced items is counted.

COMPARISON OF THEORETICAL STOCKS (STOCK CARDS & ISYSTOCK)
Each month a comparison of the stock must be done between stock cards and Isystock before the
monthly inventory (at least for the inventoried items).
The first time this will be heavy and time-consuming work, but it will save time and make the
inventory more fluid in the future.
For items with a difference, the movements need to be compared and you need to search for an
explication.

Analysis

The different sources of error need to be detailed; the percentage linked to the total number of
differences will be calculated automatically. Like that, you can take appropriated measures to
improve the next inventory.

INVENTORIED ARTICLES

Analysis

The different causes of errors need to be detailed; the percentage linked to the total number of
differences will be calculated automatically. Like that, you can identify the main causes of errors and
take appropriated measures to improve the next inventory.

After Analysis

This represents the number of the remaining items on the inventory report with remaining,
unexplainable discrepancies between the physical stock and the value on the stock card.
The attached inventory report must show these items with the valued difference.
