Storage on KITS

All products stored in the pharmacy should be on shelves. Calculated using the
number of items or 50 cm3/item, which is equivalent to 16 items per four-shelf
shelving unit.
Metal shelving units, dimensions: 2 x 0.6 x 1.75 m; avoid wood because of
termites and because it is hard to clean. Distance between shelving units: 1.20m.

/
/

Example: storing 100 items requires 7 shelving units, and thus 21 m3 of floor
space.
